# 🛒 SauceDemo Automation Framework

## 📌 Project Overview
This project is a Selenium automation framework developed to test the checkout flow of the SauceDemo application.

It follows the Page Object Model (POM) design pattern and uses TestNG for test execution.

---

## 🚀 Tech Stack
- Java
- Selenium WebDriver
- TestNG
- Maven
- Extent Reports

---

## 📂 Project Structure
src/main/java
- base → WebDriver setup
- pages → Page classes (POM)

src/test/java
- tests → Test cases

utils → Reporting utilities

---

## ✅ Test Scenarios Covered
- Login with valid credentials
- Add product to cart
- Navigate to cart
- Checkout process
- Verify checkout completion

---

## ▶️ How to Run
1. Clone the repository
2. Open in IntelliJ / Eclipse
3. Run testng.xml

---

## 📊 Reports
Extent Report is generated after execution in test-output folder.
